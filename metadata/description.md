nano # Cloud Game / CloudRetro

Cloud Game é um serviço open-source de cloud gaming para jogos retrô.

Este app foi adaptado para rodar no Runtipi usando Docker Compose com suporte ao serviço principal e ao Xvfb.

## Pastas persistentes

- `cores`: cores/emuladores
- `games`: jogos/ROMs

## Portas

- Interface web principal: 8000
- Porta adicional: 9000
- WebRTC UDP: 8443/udp
